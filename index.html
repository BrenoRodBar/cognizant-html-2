1° JavaScript é uma linguagem de programação usada por desenvolvedores para fazer páginas interativas da Internet. As funções de JavaScript podem melhorar a experiência do usuário durante a navegação em um site, como, por exemplo, desde a atualização do feed na página da mídia social até a exibição de animações e mapas interativos. Como uma linguagem de script do lado do cliente, ele é uma das tecnologias principais da World Wide Web. Por exemplo, ao navegar na Internet, é possível visualizar a qualquer momento um carrossel de imagens, um menu suspenso “clicar para visualizar” ou mesmo mudar dinamicamente as cores dos elementos de uma página da Web. Tudo isso graças ao JavaScript.
2° Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi criada por Guido van Rossum em 1991 e atualmente é uma das linguagens de programação mais populares do mundo. Uma das razões para essa popularidade é a sua sintaxe clara e fácil de entender, o que facilita o aprendizado para programadores iniciantes. A linguagem é amplamente utilizada para diversas aplicações, desde desenvolvimento web até análise de dados, aprendizado de máquina, inteligência artificial, ciência de dados, automação e muitas outras áreas. A grande comunidade em torno do Python contribui para um ecossistema rico de bibliotecas e frameworks, como Django para desenvolvimento web, Pandas e NumPy para análise de dados, e TensorFlow e PyTorch para aprendizado de máquina.
3° Java é uma linguagem de programação de alto nível, orientada a objetos e projetada para ser portátil, o que significa que código escrito em Java pode rodar em qualquer dispositivo que possua uma Máquina Virtual Java (JVM, na sigla em inglês). A linguagem foi desenvolvida pela Sun Microsystems, que posteriormente foi adquirida pela Oracle Corporation. O Java foi apresentado ao público pela primeira vez em 1995 e rapidamente ganhou popularidade, especialmente no desenvolvimento de aplicações empresariais. Uma das principais características do Java é o seu modelo de "escreva uma vez, execute em qualquer lugar" ("write once, run anywhere" ou WORA, em inglês). Isso é possível porque o código Java é compilado para um formato de byte-code que é interpretado pela JVM. Isso permite que o mesmo código Java funcione em qualquer dispositivo que tenha uma JVM compatível, independente do sistema operacional ou da arquitetura do hardware.
4° TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft que é um superconjunto do JavaScript. Isso significa que qualquer código JavaScript válido também é um código TypeScript válido. A principal adição que o TypeScript traz ao JavaScript é um sistema de tipos estáticos opcional. Isso permite que você adicione anotações de tipo ao seu código JavaScript, o que pode ser muito útil para detecção de erros em tempo de compilação, documentação e para melhorar a qualidade do código em projetos maiores e mais complexos. O TypeScript é compilado para JavaScript puro, o que significa que o código TypeScript pode ser executado em qualquer ambiente que execute JavaScript. Isso torna o TypeScript uma escolha popular para o desenvolvimento de aplicações web do lado do cliente, bem como do lado do servidor usando Node.js.
5° C# (pronuncia-se "C Sharp") é uma linguagem de programação de alto nível, orientada a objetos e fortemente tipada, desenvolvida pela Microsoft como parte da plataforma .NET. Foi criada por Anders Hejlsberg e lançada em 2000. A linguagem foi projetada para ser moderna, simples e de propósito geral, e é frequentemente usada para desenvolver aplicações Windows, jogos com o Unity, aplicações web através do ASP.NET, e até mesmo para desenvolvimento mobile com o Xamarin.
